# Task-Management
A real product used by teams

Users
. Can register & login
. Have roles: users, admin
. Belong zero or more teams

Teams
. A user can create a team
. Can invite other users
. Team has tasks

Tasks
. Belong to a user or a team
. Have: title, description, status, priority, due date
. Status workflow: todo -> in_progress -> completed
. Task history should be trackable

Activity Logs
. When a task is created
. When status changes
. When assigned to someone

Tech stack
Node.js + PostgreSQL
